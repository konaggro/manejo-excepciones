En este ejercicio se trabaja el manejo de excepciones en Python utilizando estructuras como try, 
except, else y finally.
Se implementa la captura de errores comunes, como la división por cero y el ingreso de datos 
no numéricos (por ejemplo, escribir "hola" en lugar de un número). Además, se crea una excepción 
personalizada para validar que no se ingresen edades negativas.
En programas reales, uno de los errores más comunes es el ingreso incorrecto de datos por parte del 
usuario, como escribir texto en vez de números.
El manejo de excepciones es importante porque permite que el programa no se detenga de forma abrupta 
ante un error, mejorando la experiencia del usuario y facilitando la interacción con el sistema.
